# cindyjxie.github.io

Cindy J. Xie Writing Portfolio Website

This project was bootstrapped with [Create React App](https://github.com/facebook/create-react-app).

## Setup

`npm install`

When setting up this project on a new machine, run `npm install` to install all the project dependencies defined in `package.json`.

## Development

`npm start`

Runs the app in the development mode. Open [http://localhost:3000](http://localhost:3000) to view it in your browser.

`npm test`

Launches the test runner in the interactive watch mode.

## Deploy

### `master` branch for React App

The React App version of the website is committed to the `master` branch of the repo `cindyjxie.github.io`.

Go through these steps when you make changes to the React source code:
1. Pull the most recent commits from `origin/master`.
2. Make your changes and commit them to `master`.
3. Push your changes to `origin/master`.

### `gh-pages` branch for static build

The static version of the website is committed to the branch `gh-pages`.

To deploy a static version of the React App website to Github Pages, run:

`npm run deploy`
